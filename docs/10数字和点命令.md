# 10 数字和点命令的威力

## 数字

- operate + 数字 + 动作（范围），举例：`d3w` 删除单个单词，`d3j` 删除当前行和下面 3 行
- 数字 + operate + 动作（范围）也可以，举例： `5j` 向下移动 5 行
- 好处：被记录为一条操作，撤销方便
- 坏处需要计算数字

## 点命令

`.` 命令可以重复上一次的修改，包括

- 做了更新（增删改）
- 离开插入模式之前的一整个按键操作记录，比如 `d3w`

示例：

- `daw` 操作可以完美用 `.` 命令复用，来一个个删除单词和后面的空格
- 加分号 `jA;`，然后一键移动 `j`，一键操作 `.`

技巧：

能用 `.` 操作就不用数字，以免计算
